# 任务模块接口


## 新增任务

接口名： `/api/task/add`  
接口参数：

	*taskName // 任务名
	taskDetail // 任务详情
	taskImgUrl // 任务图片
	projectDescription // 项目描述
	projectComment // 项目备注
	testRequirement // 测试需求
	taskFileUrl // 文件地址（供下载）

请求方法： `POST`  

请求结果：  

	// 成功时
	{
	  "status": 0,
	  "message": "新增任务成功",
	  "data": {}
	}
	
	// 失败时
	{
	  "status": -1,
	  "message": "任务名不能为空",
	  "data": {}
	}
	
	// 未登录时
	{
	  "status": -1,
	  "message": "请先登录",
	  "data": {}
	}
